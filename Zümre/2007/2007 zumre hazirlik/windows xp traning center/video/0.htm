Chapter 20: Working with Video


Chapter Introduction

Computers have been able to handle video data for decades--after all, when you use a computer you are already sitting in front of a video screen. Windows XP supports video output on the screen, and video input if you add the necessary hardware to your computer. This chapter describes what formats video data is stored in, how to play video files using Windows Media Player, and how to make your own video files using the Windows Movie Maker program. The new version of the Windows Media Player that comes with Windows XP can also play DVDs.
