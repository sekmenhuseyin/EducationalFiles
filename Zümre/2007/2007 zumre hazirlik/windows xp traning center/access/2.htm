Chapter 16: Accessibility Options


Installing Accessibility Options

Most of Windows accessibility options are found on the Accessibility Options dialog box. Choose Start | Control Panel | Accessibility Options, and then click the Accessibility Options icon.

You can also use the Accessibility Wizard to guide you through the accessibility settings that make using a computer easier for you. All the accessibility options changed by the Accessibility Wizard appear on the Accessibility Options dialog box and are covered in detail in the rest of this chapter. To run the Accessibility Wizard, choose Start | All Programs | Accessories | Accessibility | Accessibility Wizard. Or, choose Configure Windows To Work For Your Vision, Hearing, And Mobility Needs on the Accessibility Options screen of the Control Panel.
