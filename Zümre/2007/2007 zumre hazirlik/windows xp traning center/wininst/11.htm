Appendix A: Installing or Upgrading to Windows XP


Installing Additional Programs

The Windows XP CD-ROM contains a number of additional programs. You can install a few of them by running the Setup program on the CD and choosing Perform Additional Tasks from the Welcome To Microsoft Windows XP window. You can choose the following:

  • Set Up Remote Desktop Connection
  • Set Up A Home Or Small Office Network
  • Browse This CD, to see the files on the CD in an Explorer window

The Valueadd folder on the CD contains more programs. Valueadd\Msft contains Microsoft programs, and Valueadd\3rdparty contains programs from other companies.
