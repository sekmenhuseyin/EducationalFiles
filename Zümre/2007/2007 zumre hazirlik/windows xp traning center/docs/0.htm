Chapter 17: Working with Documents in Windows XP


Chapter Introduction

Although word processing isn't glamorous, it is, and probably forever will be, one of the most popular uses for a computer. Windows XP comes with two tools for working with text documents--the first is the unsophisticated Notepad, and the second is the surprisingly powerful WordPad. This chapter discusses both.

This chapter also covers two other useful utilities: Calculator and Character Map. The Calculator provides all the scientific functions you may need from a calculator, and Character Map gives you access to a variety of special characters.
