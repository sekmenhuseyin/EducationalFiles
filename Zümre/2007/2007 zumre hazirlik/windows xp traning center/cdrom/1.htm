Appendix B: About the CD-ROM


Reading the E-book

The CD-ROM in the back of the book doesn't contain software--it doesn't have to, because Windows comes with a Web browser. Instead, it contains only the HTML-formatted pages of the e-book. Follow these steps to read the e-book:

  1. Put the CD-ROM into your CD-ROM drive. If your drive's autorun feature works, your Web browser (usually Internet Explorer) runs and you see the title page for the e-book. Skip to step 3.
  2. If the e-book title page doesn't appear, run Windows Explorer and display the contents of the root folder of the CD-ROM. Click or double-click the filename Winxptcr.htm (single- or double-click depending on how your desktop is configured. Your browser runs and displays the title page for the e-book.
  3. Click links as usual to browse the contents of the e-book.

If you want to read the e-book pages in the same order in which they appear in the printed book, click the Next link at the top of each Web page to move to the next Web page in the chapter. The Previous link takes you to the previous Web page in the chapter.
