Virtual CD ROM Driver 1.0

Problem : Elinızde bulunan multımedi CD leri verimli bir sekilde kullanmak istiyorsunuz. Ancak bilgisayarlarinizin hepsinde CD ROM Driver yok. Butun ogrencilerin bu CD leri ayni anda kullanmalarini istiyorsunuz.

Yanlis Cevap : CD leri her ogrencinin hardiskine kopyalarim. Herkes istedigi CD yi kullanir.

Dogru Cevap : Hardiski buyuk olan makineye Virtual CD ROM Driver 1.0 programini kurarim. Sonra herbir CD yi bu makineye bu programi kullanarak kopyalarim. Bu program her bir CD icin bir CD-Rom Driver (  E, F, G ... ) adi olusturu. Sanki  bilgisayarinizda gercekten CD Rom driver varmis gibi olur.

  Bunlari network e acarsaniz ayni anda butun ogrenciler istedikleri CD yi kullanir.

Ayrica CD yi ana makineye kopyalarken bu program compress yapar ve yerden tasarruf da saglar.
