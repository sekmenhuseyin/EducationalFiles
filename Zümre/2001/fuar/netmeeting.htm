Microsoft Netmeeting 3.1

Equipments : 2 PC , network connection , 2 Speaker, 2 Microphone, 2 Desktop Cameras, Windows 98 ME, Netmeeting 3.1

 How to Work : Bütün network bağlantıları yapılıp, iki bilgisayar açıldıktan sonra Netmeeting 3.1 çalıştırılır.

Makinelerden biri USER1 diğeri USER2 olsun. 

USER1 adlı makinede Host Netmeeting komutunu çalıştırın.

USER 2 adlı makinede CALL butonuna basarak USER1 I arayın

  Bağlantı kurulduktan sonra aşağıdaki komutlar kullanılarak : Chat, bir programı ortak kullanma, resim çizme programı olarak whiteboard programını ortak kullanma ve dosya gönderme işlerini yapabilirsiniz.
