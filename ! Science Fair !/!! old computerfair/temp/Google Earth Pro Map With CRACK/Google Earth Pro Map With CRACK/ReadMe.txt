How to:

1) . Install GoogleEarth Pro 
2) . Go to GoogleEarth Pro directory (e.g. C:\Program Files\Google\Google Earth Pro)
3) . Copy (replace) base.dll with file from the archive.
4) . Copy file "kh20" from the archive to the GoogleEarth Pro directory.
5) . Copy "default_lt.kvw" from the archive the /kvw directory (C:\Program Files\Google\Google Earth Pro\kvw).
6) . Enjoy. Now you can make movies, draw lines and polygons, save and print Premium quality images (4800 dpi) and use GPS.
